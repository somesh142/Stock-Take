SUPABASE SETUP · 3 MINUTES · FREE FOREVER

1
Create a free Supabase project
Go to supabase.com → Sign up free → New Project → give it any name → wait ~1 min for it to spin up.
2
Create the database tables
In your project: click SQL Editor in the left sidebar → click New Query → paste this and click Run:

create table if not exists scans ( id bigint generated always as identity primary key, code text unique, name text, category text, expected numeric, bc1 text, bc2 text, bc3 text, bc4 text, tax text, expiry text, qty numeric, "user" text, scan_date text, scan_time text, timestamp bigint ); create table if not exists inventory ( code text primary key, name text, category text, expected numeric, bc1 text, bc2 text, bc3 text, bc4 text, tax text ); create table if not exists config ( key text primary key, value text ); insert into config values ('admin_user','admin'),('admin_pass','admin123') on conflict (key) do nothing; alter table scans enable row level security; alter table inventory enable row level security; alter table config enable row level security; create policy "public" on scans for all using (true) with check (true); create policy "public" on inventory for all using (true) with check (true); create policy "public" on config for all using (true) with check (true);
3
Paste your project URL & anon key
Go to Project Settings → API → copy Project URL and anon public key.
